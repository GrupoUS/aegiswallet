# AEGIS WALLET - Project Snapshot

## Project Overview
**Project Name**: Aegis Wallet  
**Type**: Cryptocurrency/Digital Wallet Application  
**Tech Stack**: Vite + React + TypeScript + shadcn-ui + Tailwind CSS  
**Platform**: Lovable.dev project  
**Status**: Active Development  

## Current Architecture

### Frontend Framework
- **Core**: React 18 with TypeScript
- **Build Tool**: Vite for fast development and optimized builds
- **UI Framework**: shadcn-ui components for consistent design system
- **Styling**: Tailwind CSS for utility-first styling approach

### Project Structure
```
aegiswallet/
├── src/              # Source code
├── public/           # Static assets
├── supabase/         # Database and backend config
├── components.json   # shadcn-ui configuration
└── tailwind.config.ts # Tailwind configuration
```

### Key Features & Functionality
- Digital wallet management interface
- Secure transaction handling
- Modern React-based user interface
- Responsive design with Tailwind CSS
- Integration with Supabase for backend services

### Database & Backend
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Real-time**: Supabase real-time subscriptions
- **Storage**: Supabase Storage for file management

### Development Environment
- **IDE Support**: VS Code with TypeScript support
- **Linting**: ESLint configuration
- **Package Manager**: npm/bun support
- **Dev Server**: Vite development server with HMR

## Current Development Focus
- Building secure wallet functionality
- Implementing user authentication and authorization
- Developing transaction management features
- Ensuring responsive and accessible UI

## Dependencies & Integrations
- Lovable.dev platform for deployment and hosting
- Supabase for backend services

- Tailwind CSS for styling framework
